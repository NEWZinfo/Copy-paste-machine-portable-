# Copy-paste-machine-portable-
The Portable Copy-Paste Machine is a custom-built device that captures handwritten or printed content and reproduces it onto another page. Using dismantled printer components and microcontrollers like Arduino and ESP32, it reduces manual copying, saving time and improving accuracy for students and documentation work.
